# 基于STM32+FPGA的模拟电路信号数字重建系统

本项目实现了一个基于STM32与FPGA协同工作的数字重建系统，能够对未知的模拟电路进行信号采集、处理、分析和重建。系统可记录电路的幅频、相频响应特性，对多种输入信号（正弦波、三角波、矩形波等）经过未知电路后的输出均能较好地还原输出，连续、同频、稳定显示，一定程度上能替代原电路。

## 项目内容
- 协同系统：STM32负责控制与分析，FPGA负责高速信号处理与输出。
- 完成信号的采集、特征提取、分析与数字重建。
- 适应多种输入周期信号，输出稳定、连续。

## 主要工作
- 负责FPGA侧架构设计与Verilog实现，与STM32配合完成信号的学习、建模和重建输出。

## 系统流程
1. 学习阶段：
   - 扫频激励未知RLC电路，采样输出，FFT分析，识别电路类型，提取参数。
   - STM32控制频率步进，FPGA采样并处理，通过SPI传输数据。
2. 重建阶段：
   - 根据建模结果生成波形查找表，FPGA高速输出，实现未知电路输出复现。
